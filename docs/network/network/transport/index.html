# 传输层

传输层提供了主机应用程序进程之间的端到端的服务

# 基本功能

  • 依据端口号寻址
  • 数据分隔与重组
  • 连接管理
  • 流量控制
  • 数据纠错

# 常见协议

TCP：传输控制协议，面向连接

UDP：用户数据报协议，无连接
