# 管理页面

管理界面是为开发者准备的，用于自动根据模型创建后台界面

## 创建账号

```py
py manage.py createsuperuser
```

## 注册应用

```py
# [app_name]/admin.py
from django.contrib import admin

# 这里假设 models 中配置了 AppModel 模型类
from .models import AppModel

admin.site.register(AppModel)
```

## 管理模型

```shell
py manage.py createmigrations [app_name]
py manage.py migrate
py manage.py runserver
```

进入 http://127.0.0.1:8000/admin/ 页面

此时可以看到应用中注册的模型

可以通过该后台进行数据的管理
