# 影响消费者的因素

1. 消费者的需求是什么
2. 群体印象
   - 色彩
   - 冷暖
   - 整齐与秩序
3. 群体影响
4. 网络外部性（用户越多，效用越高）
   - 产品特征
     - 用户之间点对点的联系 (如 UGC 平台)
     - 用户被关键节点所联系 (如 银行、电信运营商、交通运输)
   - 行业特性
     - 先发优势
     - 强者恒强，赢家通吃
   - 用户选择
     - 转换成本高（生态绑定）
     - 可替代项少（龙头垄断）
5. 价格(影响消费者误判)
   - 规格变化(增大商品内容而非价值，更容易消费者购买)
   - 大数化小(降低消费者对大数字的敏感程度，如 分期付款、按日均摊)
6. 收入(影响消费者误判)
   - 金融工具
